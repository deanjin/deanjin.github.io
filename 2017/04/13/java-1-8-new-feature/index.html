函数接口

只有一个方法的接口叫函数式接口，java1.8提供了一个新注解@FunctionalInterface。
java1.8在Callable,Runable,Comparator等接口中增加了@FunctionalInterface注解

默认方法

接口可以构造一个默认方法，继承这个接口可以选择不重写这个方法，此时，采用默认方法

当两个接口采用了两个同名的默认方法，且类A同时继承了这两个接口，此时为了避免多重继承，类A需要重写这个方法

stream

stream = 函数式编程+并发计算，发挥多核优势

intermediate: 输入一个stream，经过转换，输出一个新的stream, 此时为惰性求值，并没有遍历steram。操作有：filter, map, flatMap,distinct,skip,limit,sorted
terminal:输入一个stream，计算结果。操作有: foreach, sum, count, max, min, collect, reduce
short-circuting: 满足一个条件即中断。操作有：findAny, matchAny, firstFind, limit, allMatch, noneMatch
