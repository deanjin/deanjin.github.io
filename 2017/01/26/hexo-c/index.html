创建HEXO博客

创建方法

一个不错博客平台，nodejs, 可以通过github提供公网访问，关键是支持macdown！
本质上就是一个git项目，把本地静态文件上传到github，通过域名映射提供对外访问能力，所以目前是静态web服务。

详细搭建过程参照下面链接(搭建过程中槽点很多，基本上下面文章都提到了)：

一些坑点

  • node-sass 安装卡在 node scripts/install.js

如遇到这个问题：node-sass 安装卡在 node scripts/install.js 解决办法，请参照下面链接

  • 如果运行hexo命令出现下面报错：
    { [Error: Cannot find module ‘./build/default/DTraceProviderBindings’] code: ‘MODULE_NOT_FOUND’ }

该命令实际上不影响运行，请运行：
npm install -g hexo-cli --no-optional

(作者也不太清楚问题原因，网上有很多解决方案，经测试，貌似只有这个方案是在我机器上是可行)
